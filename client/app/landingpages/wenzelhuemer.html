Project description

The project is an WEB-APP which uses spotify data to provide the user with infos about personal music categories or top artists.
The data gets provided by the spotify api where you can authenticate users via access tokens.

Features

  • Show spotify profile info and top genre
  • Show top artists of last few weeks (with popularity, name and genre)
  • Show song infos like acousticness, energy or danceability (also with radar chart)
  • Timeline of last 50 songs with a line chart
  • Create playlist of recent top tracks

Lessons learned

  • Some new libraries (like chart.js or the parcel-bundler)
  • Importance of well structured CSS (Different files like theme.css or font_baseline.css)
  • General structure of a client & server web project
  • Awesomeness of css grid layouts
  • Difficulty of meaningful naming
  • Frameworks can be extremely helpful and save a lot of time (like Angular)
