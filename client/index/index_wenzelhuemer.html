Project description

The project is an web application for Spotify.
With the web app it is possible to get a detailed view of your spotifiy history by using the spotify api.

Technologies and libraries

Backend (API)

  • The server is implemented with Node-JS
  • It uses Typescript.

Frontend (Website)

  • The website is implemented with simple HTML & CSS.
  • It also uses Typescript.
